# Roadmap (indicative)

Pistes d'évolution possibles pour le configurateur, partagées à titre indicatif. N'engagent pas de date de livraison.

- [ ] Export de la composition du bracelet en image PNG/PDF partageable
- [ ] Sauvegarde de compositions multiples par utilisateur (compte / lien partageable)
- [ ] Mode comparateur de compositions (côte à côte)
- [ ] Internationalisation (EN, ES) de l'interface et de la FAQ
- [ ] Amélioration de l'accessibilité (navigation clavier complète, ARIA sur le configurateur SVG)
- [ ] Statistiques anonymisées des combinaisons de pierres les plus demandées (sans donnée personnelle)
- [ ] API publique de lecture du catalogue de pierres (nom, propriétés, couleur) à but pédagogique

## Comment proposer une idée

Ouvrez une [issue](../../issues) avec le label `enhancement` en décrivant le besoin utilisateur derrière la fonctionnalité, pas seulement la fonctionnalité elle-même.
